How It Works

ChaiTailwind follows a clean 4-stage pipeline to transform classes into styles.

🔍

1. Scan

Traverses the DOM and finds all elements with class attributes.

scanDOM(document)
→
🧩

2. Parse

Extracts chai-* classes and splits them into utility + value.

chai-p-24 → {utility: "p", value: "24"}
→
🗺️

3. Map

Converts the parsed utility into a CSS property-value object.

{padding: "24px"}
→
🎨

4. Apply

Sets inline styles on the element and removes the processed classes.

element.style.padding = "24px"
